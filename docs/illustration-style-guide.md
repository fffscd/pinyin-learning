# 拼音插画风格规范

## 画布与结构

- 统一使用 `viewBox="0 0 120 120"`。
- 单主体居中，主体宽高占画布约 55%–80%，不在 SVG 内绘制卡片背景框。
- 使用 SVG 基本图形与 path，不使用滤镜、外链、位图和嵌入字体。
- 单幅插画控制在 30 个可见节点以内。

## 线条与配色

- 主描边使用 `#223047`，默认宽度 `5`，端点与连接均为圆角。
- 主体色优先使用条目 `color`，可搭配：`#ffe9a6`、`#ffb8a4`、`#dff5df`、`#dff2ff`、`#ffffff`。
- 颜色需要在白底和浅色卡片背景上保持清楚轮廓，不使用低对比度细节表达关键语义。

## 构图与语义

- 图像直接对应条目的 `word` 例词，优先画幼儿熟悉的具体名词。
- 多义词使用 4–5 岁儿童更常见的义项；同义物体保持形态一致。
- 不依赖文字、数字或复杂背景才能辨认目标物体。
- 同批 10–15 幅完成后，在 `.tmp/illus-preview.html` 与 emoji 临时图并排检查。

## 验收

1. 至少 3 名成人独立看图说词，目标词或明确近义词达到一致。
2. 在桌面和 375px 窄屏检查轮廓、裁切和对比度。
3. 运行 `node scripts/report-missing-illustrations.js` 更新 backlog。
4. 运行 `node scripts/verify-child-shell.js`，确认儿童壳层与配图题未回退。

图像语义清晰度优先于装饰细节。无法稳定说出目标词的图不进入 `ILLUSTRATION_IDS`。
